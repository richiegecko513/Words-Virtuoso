New dates

Write a program that searches a text for dates in two different formats: yyyy.mm.dd and yy.mm.dd. The search results should be printed one per line.

Don't forget about \b. The format yyy.mm.dd isn't correct!


Sample Input:
Date 1: 2022.06.06 Date 2: 2021/01/01; date 3: 20.02.22

Sample Output:
2022.06.06
20.02.22



Memory limit: 256 MB
Time limit: 8 seconds
